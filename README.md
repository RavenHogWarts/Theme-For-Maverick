# Kepler

A theme for [Maverick](https://github.com/AlanDecode/Maverick), designed for wiki site.

**Usage**

Set `template` in Maverick configuration file as bellow:

```python
template = {
    "name": "Kepler",
    "type": "git",
    "url": "https://github.com/RavenHogWarts/Theme-For-Maverick.git",
    "branch": "Kepler"
}
```
