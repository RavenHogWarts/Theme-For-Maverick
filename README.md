# Galileo

A theme for [Maverick](https://github.com/AlanDecode/Maverick), designed for personal blog.

**Usage**

Set `template` in Maverick configuration file as bellow:

```python
template = {
    "name": "Galileo",
    "type": "git",
    "url": "https://github.com/RavenHogWarts/Theme-For-Maverick.git",
    "branch": "Galileo"
}

template = {
    "name": "Galileo",
    "type": "local",
    "path": "../Galileo"
}
```
